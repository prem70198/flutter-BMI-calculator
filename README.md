## 📐 BMI Formula

BMI is calculated using:

```
BMI = Weight (kg)
/ Height² (m²)
```

Example

```
Weight = 70 kg

Height = 1.75 m

BMI = 70 / (1.75 × 1.75)

BMI = 22.86
```

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/prem70198/flutter-BMI-calculator.git
```

Move into the project

```bash
cd flutter-BMI-calculator
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```

| BMI | Category |
